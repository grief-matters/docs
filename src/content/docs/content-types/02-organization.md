---
title: "Organization"
---

The **Organization** content type is a singleton content type that encapsulates content related to the Why Grief Matters organization.

:::note[Singleton]
If a content type is a **singleton** it means there can only be one.
:::

### Available Fields

- **Name**
- **Logos**
- **Mission Statement**
- **Slogan**
- **Core Values**
- **Small Print**
